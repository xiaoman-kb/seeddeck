# Security Policy

Please report security issues privately before publishing details. Use GitHub's private vulnerability reporting if it is enabled for the repository; otherwise, contact the maintainer through the repository owner profile and share only the minimum information needed to reproduce the issue.

Do not use public issues for vulnerability reports involving arbitrary file access, protocol handling, torrent metadata parsing, package supply chain concerns, or command execution.

SeedDeck does not host or distribute content. It searches public third-party sources and connects to the BitTorrent network from the user's machine. Please use it only for content you are legally allowed to download or share.
